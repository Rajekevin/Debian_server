## Configuring web server in Debian 9
### Update Package
```bash
apt-get update
apt-get upgrade
```

### Http Serveur
```bash
apt-get install apache2
```

### PHP
```bash
apt-get install php
```
### MySqlServer
```bash
apt-get install mysql-server
```

### PhpMyAdmin
```bash
apt-get install phpmyadmin 
```
